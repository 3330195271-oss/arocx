常见问题

这些问题通常会决定团队是否愿意开始用。

对设备租赁团队来说，软件是否好用往往要过三关：能不能装、数据能不能同步、团队能不能协作。下面这些回答会更直接一点。

ARO 支持 Windows 和 Mac 吗？

支持。官网会优先展示当前可下载安装的版本，常见办公场景下可以直接按自己的系统选择安装包。

订单和设备信息会同步吗？

会。订单、设备和相关状态会同步到云端，方便同一个账号或团队成员在不同设备上继续查看和处理。

可以把订单同步到飞书共享表格吗？

可以。适合希望在软件里推进流程，同时继续在飞书里共享订单状态和设备信息的团队。如果你要实际开始绑定，建议直接看飞书接入教程。

飞书接入时最容易漏掉什么？

最容易漏掉的是：没有创建企业自建应用、权限开了但没发布版本、以及只填了 App ID 和 App Secret 却还没自动建表或手动填写 App Token 与 Table ID。

软件会自动检查新版本吗？

会。软件会定期检查可用更新，官网也会同步显示当前版本、下载入口和更新公告。

适合哪些设备租赁团队？

比较适合摄影器材、活动设备、仓储周转设备等需要同时管理订单、库存、发货和租期的团队。

为什么要把起租日、到期日和发货日放进订单列表？

因为这三个时间点直接决定当天的执行动作。能直接看到它们，排期、发货和催还都会更顺。
